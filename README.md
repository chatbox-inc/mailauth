## Usage

sample フォルダを中心に

- 認証トークンと、ユーザのテーブルモデルとを作成
- ServiceProviderを作成


## 設計方針

Controllerは基本いじらない方向でServiceProviderを組む
(そもそもパッケージの処理内容がApiAuthにコントローラを足しているだけなので)
